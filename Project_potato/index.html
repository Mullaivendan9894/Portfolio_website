Potato Disease Classification Using Deep Learning

Project Overview

The goal of this project is to develop an automated system for classifying potato leaf diseases using deep learning. Potato plants are susceptible to diseases like Early Blight and Late Blight, which can significantly impact crop yield and quality. Early detection of these diseases is crucial for farmers to take timely action and minimize losses. This project leverages Convolutional Neural Networks (CNNs) to classify potato leaf images into three categories:

  • Early Blight
  • Late Blight
  • Healthy

The model is trained on the PlantVillage dataset, which contains labeled images of potato leaves affected by these diseases. The final model can be deployed as a web application to assist farmers in identifying diseases quickly and accurately.

Key Features

  • Automated Disease Detection: The system eliminates the need for manual inspection, saving time and reducing reliance on expert knowledge.
  • High Accuracy: The CNN model achieves an accuracy of 95.83% on the test dataset.
  • User-Friendly Web Application: A Streamlit-based web app allows users to upload images and receive predictions with confidence scores.
  • Data Augmentation: Techniques like rotation, flipping, and zooming are used to enhance the dataset and improve model generalization.

Technologies Used

  • Programming Language: Python
  • Libraries/Frameworks:
    • TensorFlow, Keras (for building and training the CNN model)
    • NumPy, Pandas (for data manipulation)
    • Matplotlib, Seaborn (for visualization)
    • Scikit-learn (for evaluation metrics)
    • Streamlit (for web app development)
  • Dataset:
    • PlantVillage dataset (publicly available dataset for plant disease classification)
    • Downloaded from Kaggle

Methodology

1. Data Preprocessing

The dataset consists of 2,152 images of potato leaves, divided into three classes: Early Blight, Late Blight, and Healthy. ImageDataGenerator is used to preprocess and augment the data. Techniques like rescaling, rotation, flipping, and zooming are applied to improve model robustness. The dataset is split into:

  • Training set (70%)
  • Validation set (20%)
  • Test set (10%)

3. Model Training

The model is trained for 50 epochs using the Adam optimizer and Sparse Categorical Crossentropy loss function. Training and validation accuracy/loss are monitored to ensure the model is learning effectively.

4. Model Evaluation

The model is evaluated on the test set, achieving a test accuracy of 95.83%. A confusion matrix and classification report are generated to analyze performance.

5. Web Application

A Streamlit-based web app is developed to allow users to upload potato leaf images and receive predictions. The app displays the uploaded image, predicted class, and confidence score.

Results and Insights

The model performs exceptionally well, with high accuracy and robust generalization. Early Blight and Late Blight are classified with 99% precision and recall, while Healthy leaves are classified with 93% precision and recall. The web app provides a practical tool for farmers to detect diseases early and take preventive measures.

Future Improvements

Conclusion

This project demonstrates the power of deep learning in agriculture, specifically for disease detection in potato plants. By automating the classification process, farmers can save time, reduce crop losses, and improve yields. The high accuracy and user-friendly web app make this solution practical and impactful.
